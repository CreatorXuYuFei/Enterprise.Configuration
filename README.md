# Enterprise.Configuration
This is a universal enhanced JSON serialization plugin deeply optimized based on System.Text.Json. Compatible with all .NET versions, it is designed to fix common issues like malformed JSON and type conversion errors, and supports global reuse across projects.

# 微软依赖包：
Install-Package Microsoft.Extensions.DependencyInjection -Version 9.0.0
Install-Package Microsoft.Extensions.Options -Version 9.0.0
